# Sensor-Based Monitoring System (Internship Task 1)

## Project Overview
This project is developed as part of my internship Task 1.  
It demonstrates a basic sensor-based monitoring system using an Arduino board to read and process real-time environmental data from different sensors.

The system helps in understanding how temperature, motion, and light sensors work together in embedded systems.

---

## Components Used
- Arduino UNO  
- Temperature Sensor  
- Motion Sensor (PIR Sensor)  
- Light Sensor (LDR)  
- Breadboard  
- Jumper Wires  
- LED (for indication)  
- Resistors (220Ω, 10kΩ)

---

## Working Principle
- The **temperature sensor** measures surrounding temperature.
- The **PIR motion sensor** detects movement in the environment.
- The **LDR (light sensor)** detects light intensity.
- Based on sensor readings, the Arduino processes the input and activates output devices like LED indicators.

---

## Circuit Operation
- When motion is detected → LED turns ON  
- When light level is low/high → system responds accordingly  
- Temperature values are continuously monitored and displayed via serial output
